О проекте

Этот сервис позволяет сократить длинный URL до короткой удобной ссылки, хранит информацию о переходах и предоставляет HTTP API для интеграции с другими приложениями.

В проекте используются FastAPI, шаблоны Jinja2, работа с базой данных и токенами доступа для защищённых операций.
